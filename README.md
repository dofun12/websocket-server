# websocket-server
An Web Socket Server On Springboot without Stomp and with JS pure, and with History using H2


Running the server
````bash
mvn spring-boot:run
````

On Docker
````bash
docker run -it --rm -p 8080:8080 --name websocket-server lemanoman/websocket-server:latest
````


Available on 
````
http://localhost:8080
````
